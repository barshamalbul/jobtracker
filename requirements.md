## Problem
Job seekers struggle to track multiple job applications, statuses, and notes.

## Solution
A web-based application to track job applications with statuses and notes.

## Functional Requirements
- Add a job application
- View all applications
- Update application status
- Edit or delete an application
- Filter applications by status

## Fields
- Company Name
- Job Title
- Job Link
- Location
- Application Date
- Status
- Notes
